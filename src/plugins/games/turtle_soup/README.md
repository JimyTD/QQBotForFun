# 海龟汤 · 插件说明

- Game ID: `turtle_soup`
- 指令：
  - `/play turtle_soup` — 开始一局
  - 提问：以 `?` 或 `？` 结尾
  - 宣告：以 `汤底:` / `答案:` / `宣告:` 开头
  - `/soup status` — 查看进度
  - `/soup giveup` — 投降公布汤底
  - `/soup recap` — 回顾关键线索
  - `/quit` — 终止本局

详细设计见 [`docs/games/turtle-soup.md`](../../../../docs/games/turtle-soup.md)。
