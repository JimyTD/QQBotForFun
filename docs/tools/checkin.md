# 📅 每日签到（tools/checkin）

- **Status**: Draft v1.1
- **Last Updated**: 2026-05-15
- **Type**: 小工具（tool），**非游戏**（无对局、无输赢）

---

## 1. 定位与边界

### 1.1 这是什么

每日签到——群友每天 `@机器人 签到`，领取固定的 coin + score 奖励。
**零门槛的日常正反馈**，让不爱玩游戏的群友也能积累金币，偶尔参与需要押注的游戏。

### 1.2 这不是游戏

**显式不走 `GameBase` 框架**，理由：
- 没有对局概念（一条命令一个结果，不存状态）
- 没有输赢
- 没有多玩家交互

放在 `src/plugins/tools/checkin/` 下，作为**工具型功能**。

---

## 2. 触发方式

```
@机器人 签到
@机器人 打卡
@机器人 checkin
```

需要 `@机器人`（遵循指令铁律），`priority=3, block=True`。

---

## 3. 核心规则

### 3.1 每日一次

- 以**自然日（北京时间 UTC+8 0:00 重置）**为周期
- 同一 QQ 号当天重复签到 → 友好提示"今天已经签过啦"，不重复发放
- 判定方式：DB 记录 `last_checkin_date`（date 类型），与 `datetime.now(tz=CST).date()` 比较

### 3.2 基础奖励

| 货币 | 数量 | 设计理由 |
|---|---|---|
| **coin** | **+10** | 每天签到即可参与押注类游戏 |
| **score** | **+1** | 象征性积分，30 天 = 30 分 ≈ 1.5 局海龟汤，不碾压游戏玩家 |

### 3.3 连续签到阶梯加成

| 连续天数 | coin | score | 说明 |
|---|---|---|---|
| 1~6 天 | +10 | +1 | 基础奖励 |
| **7 天** | **+20** | **+3** | 周里程碑 |
| 8~13 天 | +10 | +1 | 基础奖励 |
| **14 天** | **+20** | **+3** | 双周里程碑 |
| 15~20 天 | +10 | +1 | 基础奖励 |
| **21 天** | **+20** | **+3** | 三周里程碑 |
| 22~29 天 | +10 | +1 | 基础奖励 |
| **30 天** | **+50** | **+5** | 满月大奖 |

- **里程碑奖励替代当天基础奖励**（不叠加）
- **断签重置为第 1 天**（不惩罚，只是加成归零，符合"永不负反馈"原则）
- **满 30 天后循环重置**（从第 1 天重新开始计数）

### 3.4 签到反馈消息（两条消息）

签到时发送**两条消息**：
- **第一条**：签到结果（金币、积分、连续天数）
- **第二条**：今日运势 = 名人寄语（配梗图）+ 黄历宜忌（纯文字）

#### 第一条 · 签到结果

签到成功：
```
📅 签到成功！
━━━━━━━━━━━━━━━━━━
连续签到：第 7 天 🔥🔥
💰 金币 +20（余额 185）
🏆 积分 +3（余额 47）
🎁 连续 7 天里程碑奖励！
📌 再签 7 天达成 14 天里程碑
─────────────────
累计签到 42 天
```

重复签到：
```
📅 今天已经签过啦～
━━━━━━━━━━━━━━━━━━
连续签到：第 7 天 🔥🔥

明天记得再来哦！
```

#### 第二条 · 今日运势（仅签到成功时发送）

同时包含名人寄语和黄历宜忌：

**名人寄语风格**（配梗图）：
```
[名人梗图]
🗣️ 马斯克寄语：
"我很欣赏你这种每天准时打卡的人，考虑来 SpaceX 当倒计时员吗？"
```

**黄历宜忌风格**（纯文字）：
```
📜 今日黄历 ─── 中吉
宜：摸鱼、喝奶茶、发表情包
忌：碰 CSS、说"这把稳了"、立 flag
```

## 4. 数据存储

### 4.1 表 `tool_checkin`

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | BigInt PK | 自增 |
| `qq_id` | BigInt | QQ 号 |
| `last_checkin_date` | Date | 最后签到日期（北京时间） |
| `streak` | Int | 当前连续签到天数 |
| `total_checkins` | Int | 历史总签到次数 |
| `created_at` | Timestamp | 首次签到时间 |
| `updated_at` | Timestamp | 最后更新时间 |

唯一索引：`ux_tool_checkin_qq (qq_id)`

### 4.2 经济入账

通过 `core.economy.add()` 直接调用（不走 `GameBase.award`，因为这不是游戏）：

```python
await economy.add(qq_id, coin, reason=f"checkin:day{streak}", currency="coin")
await economy.add(qq_id, score, reason=f"checkin:day{streak}", currency="score")
```

reason 带 `checkin:` 前缀，流水可追溯。

## 5. 代码结构

```
src/plugins/tools/checkin/
├─ __init__.py        # 插件入口，注册 models + commands
├─ commands.py        # /签到 命令处理器（发两条消息）
├─ models.py          # tool_checkin 表定义
├─ storage.py         # DB 读写（查签到记录、更新连续天数）
├─ service.py         # 核心业务逻辑（签到判定、奖励计算）
└─ fortune.py         # 今日运势（名人寄语 + 黄历宜忌）

resources/checkin/
└─ celebrities/       # 名人梗图（每人一张，如 trump.jpg、musk.jpg）

scripts/crawler/
└─ celebrity_meme_downloader.py  # Bing 图片爬虫，下载名人梗图候选
```

## 6. 与现有系统的关系

| 维度 | 说明 |
|---|---|
| **经济系统** | 直接调用 `core.economy.add()`，reason 带 `checkin:` 前缀 |
| **排行榜** | score 自然进入 `/榜` 排行榜（与游戏积分共用） |
| **个人资料** | `/资料` 展示连续签到天数和累计签到天数 |
| **防刷** | 每日一次硬限制 + economy 流水表审计 |
| **CLI** | 不需要 CLI adapter（不是游戏） |

## 7. 数值平衡

| 来源 | 每次 coin | 每次 score | 说明 |
|---|---|---|---|
| 签到（基础） | 10 | 1 | 日常 |
| 签到（7/14/21 天） | 20 | 3 | 周里程碑 |
| 签到（30 天） | 50 | 5 | 满月大奖 |
| 海龟汤（赢一局） | 100 | 20 | 对比参考 |
| 趣味问答（一局均值） | ~17 | ~28 | 对比参考 |

签到产出远低于游戏产出，但足以让非游戏玩家每天有金币参与押注类游戏。

## 8. 今日运势附加功能

### 8.1 概述

签到成功后，额外发送一条**今日运势**消息。这是一个**非持久化**的纯趣味功能（抽个签而已），不影响经济系统，不与其他插件耦合。

每次签到**同时包含**以下两种内容：

| 风格 | 内容 | 是否配图 |
|---|---|---|
| **名人寄语** | 名人说沙雕话，反差幽默 | ✅ 配名人梗图 |
| **黄历宜忌** | 搞笑宜忌条目，纯娱乐 | ❌ 纯文字 |

#### 组合规则

- 每次随机抽取 1 个运势等级
- 从"宜"池随机抽 3 条（不重复）
- 从"忌"池随机抽 3 条（不重复）
- 宜和忌的条目不会重叠（它们来自不同的池子）

### 8.4 实现要点

- `fortune.py` 纯函数模块，不依赖 DB，不持久化
- 名人寄语数据硬编码在 `fortune.py` 中（JSON 列表）
- 黄历宜忌数据同样硬编码
- 图片通过 `base64://` 方式发送（与 reminder 插件一致）
- 图片在启动时预缓存到内存（`_image_cache`），避免每次签到时做同步 IO
- 如果名人梗图缺失，降级为纯文字（不发图片，不报错）

### 8.5 图片获取工具

`scripts/crawler/celebrity_meme_downloader.py`：
- 使用 Bing 图片搜索，按名人名搜索梗图
- 每个名人下载 3~5 张候选图到 `resources/checkin/celebrities/<name>/`
- 人工挑选后，将最终图片重命名为 `<name>.jpg` 放到 `resources/checkin/celebrities/` 根目录
