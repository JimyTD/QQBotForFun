## 一、调研结论（Executive Summary）

### 1.1 两个 mod 各自的核心

| | 群峦传说 (TFC) | 格雷科技 (GTCEu) |
|---|---|---|
| **主轴** | believable 生存 + 三大时代手工冶金 | 电压分层 + 矿物处理网 + 工业扩张 |
| **进度感** | 石器 → 铸造 → 铁器（→ 彩钢） | 手搓 → 蒸汽 → LV → MV → HV … |
| **操作乐趣** | 制陶/坑窑/浇铸/砧锻/勘矿/营养 | 处理路线、副产物、蒸汽/EU、瓶颈材料 |
| **资源压力** | 饥渴/营养/温度/工具分级/矿脉稀少 | 硫/锡/橡胶/电路…、蒸汽供给 |
| **不能搬到 QQ** | 3D 塑形图案、实时温度场、重力 | 物流管道、电压网、产线布局 |

### 1.2 群游戏融合公式（定稿）

```
┌──────────────────────────────────────────────────────────────┐
│  TFC 层（前期～铁器）                                         │
│  生存补给 · 制陶坑窑 · 浇铸 · 勘矿 · 砧锻 · 时代跃迁          │
├──────────────────────────────────────────────────────────────┤
│  GT 层（青铜过剩～蒸汽）                                      │
│  蒸汽供给 · 粉碎/洗涤 · 路线分叉 · 副产物仓 · 砖高炉          │
├──────────────────────────────────────────────────────────────┤
│  群游戏层（持久沙盘）                                         │
│  部落仓库 · 个人工具栏 · 体力 · 矿脉枯竭 · 季节 · 群项目      │
└──────────────────────────────────────────────────────────────┘
```

**趣味等价物**（不依赖空间 simulation）：

1. 时代门槛 + 工具 tier 门票
2. 多步加工链（每步 = 面板选项，配方全透明）
3. 勘矿开脉 + 岩层匹配 + 储量枯竭
4. 同矿多种处理路线（快 vs 副产多）
5. 仓库瓶颈可视化（硫堆满、锡告急）
6. 世界随时间变（季节、腐败、脉枯）

### 1.3 明确不做

- 猜合金配比 / 隐藏合成表
- 合成格摆放、线路、上下游物流
- 一局 session、随意过期、通关删档
- 个人背包格子；金币买生存补给

## 二、群峦 (TFC) 机制库 → 群游戏译法

> 来源：[TFC 手册](https://terrafirmacraft.github.io/Field-Guide/18/zh_cn/)

### 2.1 三大时代（官方）

| 时代 | 标志能力 |
|---|---|
| **石器** | 捡石/棍、石制工具；低效 |
| **铸造** | 制陶、坑窑、浇铸；铜/青铜工具；**镐、勘矿镐、凿、锯** |
| **铁器** | 高炉、生铁、砧锻钢；→ 黑/红/蓝钢 |

### 2.2 石器时代：真实步骤 → 群游戏行动链

| Mod 步骤 | 群游戏译法 | 面板行动 |
|---|---|---|
| 捡石子/木棍 | 探索保底产出 | `🔍探索` |
| 石子塑形（凿图案） | **改为选形态**，不玩图案 | `🔨加工` → 石刃/石镐头 |
| 工具头 + 木棍 | 装柄（固定配方） | `🔨加工` → 石镐/石斧 |
| 找地表矿粒 | 探索事件「发现矿粒」 | 解锁脉点线索 |
| 找粘土（指示植物/河边） | 探索「粘土层」 | 解锁粘土采集 |

### 2.3 铸造时代：制陶链（TFC 风味核心，此前文档偏薄）

**Mod 真实链**：粘土 → 塑形 → **坑窑**（8干草+8原木，~1400°C）→ 瓷器/模具

| 物品 | 作用 | 群游戏 |
|---|---|---|
| 未烧陶 | 无效 | 加工失败提示 |
| 坩埚/小缸 | 熔炼容器 | 建造「坩埚」里程碑 |
| 锭模/镐模 | 浇铸 | 建造「模具工坊」 |
| 陶罐 | 储水、携带 | **水井前置**：陶罐 → 储水容量 ↑ |
| 坑窑 | 烧陶+熔化 | 建造「坑窑」→ 解锁 `烧窑` 行动 |

**`烧窑` 行动**（/async 友好，不用等 6 小时）：

- 每次 `烧窑` 可处理 N 件（未烧陶 / 待熔矿 / 待烤食）
- 消耗：干草、原木、（可选）木炭
- 产出：模具、熔融铜、熟肉
- **季节修正**：雨季坑窑效率 -20%（TFC 气候味道）

### 2.4 浇铸链（铸造时代主轴）

```
矿粒/矿石 ──[坑窑/坩埚·熔化]──> 熔融金属
         ──[浇铸·需模具]──> 工具头/锭
         ──[装柄]──> 铜镐/青铜镐
```

- 熔化要求：**金属量 ≥ 阈值**（Mod 为 100mB；群游戏：至少 N 个矿粒）
- 合金（青铜）：坩埚内 **配方透明** `铜×9 锡×1`，一次 `合金` 行动
- 进入铸造时代判定：**做出青铜镐**（不是只做青铜锭）

### 2.5 勘矿与矿脉（TFC 灵魂之一）

| Mod 机制 | 群游戏译法 |
|---|---|
| 地表矿粒仅在脉上方 15×35 格 | 探索发现「脉点」；未发现则无定向采矿 |
| 勘矿镐 25³ 扫描；有误报 | `勘探` 行动：回报 微量～巨量 + **可信度** |
| 不同矿石 ↔ 岩石类型/高度 | 脉点带 `rock_tag`；**岩层不匹配** → 采集 ×0.5 |
| 矿脉密度（贫瘠/普通/富集） | 脉点 `grade`：影响每次采集量 |
| 矿脉挖完 | `remaining` 归零 → 需再勘探 |

**勘矿镐 tier**：铜勘矿 → 误报率 30%；青铜 → 15%；铁 → 5%

### 2.6 铁器时代：高炉 + 砧锻

| Mod 步骤 | 群游戏 |
|---|---|
| 高炉（铁+助焊剂+木炭+鼓风） | 群项目「高炉」+ 持续消耗 **木炭** |
| 生铁锭 | 加工链产出 |
| 砧锻：生铁→高碳钢→钢 | `砧锻` 行动，**2～3 步**，见 §2.7 |
| 彩钢（黑/红/蓝） | **扩展内容**，MVP 不做 |

### 2.7 砧锻：有抉择，不猜图案

Mod 是瞄准图案锤炼；群游戏改为 **策略二选一**（每步点表情）：

| 模式 | 耗时/体力 | 材料损耗 | 适用 |
|---|---|---|---|
| **快锻** | 1 行动 | 正常 + 10% 废料 | 赶进度 |
| **精锻** | 2 行动 | 无额外损耗 | 材料紧张 |

生铁 → 钢：`快锻×2` 或 `精锻×1`（等价不同资源压力）

### 2.8 生存维度（TFC）→ 群游戏（夯实）

此前只有「部落补给池」，调研后**建议三层叠加**（仍无背包）：

#### A. 部落补给池（🍖 / 💧）

- 每次行动消耗；探索保底 +1
- 设施：陶罐储水 → 水井 → 不耗水；灶台/农田 → 不耗食物

#### B. 营养轮（简化 TFC 五元素）

| 营养 | Mod 效果 | 群游戏 |
|---|---|---|
| 果/蔬/蛋白/谷/乳 | 影响 max HP | 合并为 **均衡度** 0～100 |
| 单一食物 | 偏科降上限 | 连续吃同种采集物 → 均衡 ↓ |
| 乳制品 | 回血速度 | 均衡 > 80 → **体力恢复 +1/日** |

**行动**：`采集` 时可顺手 `备餐`（消耗采集物 → 补均衡）；不单独搞烹饪 minigame。

#### C. 季节（真实时间，/async）

| 季节 | 修正（示例） |
|---|---|
| 春 | 探索发现植物类 ↑ |
| 夏 | 💧 消耗 +1；需更多储水 |
| 秋 | 采集食物 ↑ |
| 冬 | 补给消耗 +1；坑窑需更多燃料 |

季节按 **现实周** 轮换（7 天一季），持久沙盘才有「回来发现世界变了」。

#### D. 腐败（可选，中期开启）

- 仓库 `food` 类有 `freshness`；每 N 天衰减
- **盐腌/地窖** 建筑减缓；否则浪费 → 生存压力真实存在
- 不腐败：前期简化；进入铁器时代后开启（GT 工业粮可豁免）

### 2.9 个人工具栏（无背包的「身上装备」）

TFC 多种工具 → 群游戏 **个人工具栏 6 槽**（不是物品栏）：

| 槽位 | 工具 | 作用 |
|---|---|---|
| 主手 | 镐 | 采矿 tier 与效率 |
| 副手 | 斧/铲 | 木/土采集效率 |
| 专用 | 勘矿镐 | 勘探行动 |
| 专用 | 刀 | 狩猎/剥皮（食物+皮） |
| 扩展 | 凿/锯 | 后期建筑减免材料（扩展） |
| 扩展 | 锤 | 砧锻效率 |

升级工具 = 消耗部落仓库 + `加工`；**旧工具可捐回仓库折材料**（防死档）。

## 三、格雷 (GT) 机制库 → 群游戏译法

> 来源：GTNH Steam Age、GTCEu Ore Processing

### 3.1 阶段与动力

| 阶段 | 动力 | 群游戏资源 |
|---|---|---|
| 史前 | 手搓/坑窑 | 无 steam |
| **蒸汽** | 锅炉 2S:1EU | **`steam` 蒸汽池** |
| LV+ | 电力 | 扩展：`eu` 池（MVP 可不做） |

### 3.2 蒸汽时代：不是「造个机器」，是「供能系统」

Mod 要点：锅炉缺水会炸；木管不能通蒸汽；机器有排气口。

群游戏译法：

| 概念 | 实现 |
|---|---|
| 锅炉 | 建筑；每 `添柴` 行动产 steam |
| 蒸汽池 | `steam.current / steam.cap`；面板常显 |
| 蒸汽机器 | 每次 `加工` 耗 steam；不足 → 可手搓但 ×0.5 |
| 青铜消耗 | 管道/机器大量用 bronze → **铸造时代必须过剩青铜** |

**MVP 蒸汽机器**：

| 机器 | Mod 对应 | 作用 |
|---|---|---|
| 蒸汽锻造锤 | Forge Hammer | 碎矿 1×2（初版双倍） |
| 蒸汽粉碎机 | Macerator | 粉碎矿 → 粉末链 |
| 蒸汽洗矿台 | Ore Washer | 洗涤 + 副产 |
| 蒸汽合金炉 | Alloy Smelter | 青铜更高效 |
| 砖砌高炉 | Bricked BF | 产钢 |

### 3.3 矿物处理链（GT 灵魂）

**三阶段**（面板逐步解锁）：

```
原矿 ──[粉碎/锤]──> 粉碎矿 ──[洗涤/跳洗]──> 纯净粉碎矿 ──[离心/跳过]──> 粉
                      │                      │
                   副产 A                   副产 B
```

**路线分叉**（每种矿 2 条，MVP 做 3 种矿即可）：

| 矿 | 路线快 | 路线丰副产 |
|---|---|---|
| **方铅矿** (Galena) | 粉碎 → 直接炼铅 | 洗涤 → 得 **硫** + 银痕迹 |
| **黄铜矿** (Chalcopyrite) | 粉碎 → 炼铜 | 存矿待 LV 炼 **PGM**（扩展） |
| **铁矿** | 锤碎 1×2 | 洗涤 → 硅粉/石灰粉 |

**副产物仓看板**（Pillow 专页）：硫 ×47、锡 ×3、银 ×12 —— GT 式瓶颈一眼可见。

### 3.4 GT 典型瓶颈 → 群游戏事件

| 瓶颈 | 来源 | 群游戏表现 |
|---|---|---|
| **硫** | 方铅矿/黄铁矿副产 | 橡胶/酸的前置；面板「缺硫」预警 |
| **锡** | cassiterite 脉 | 青铜/电缆；早期必勘矿 |
| **橡胶** | 橡胶树 + 硫（扩展） | LV 绝缘；探索「橡胶林」 |
| **粘土** | 地表 | 蒸汽砖高炉前置 |

### 3.5 电压与电路（扩展，非 MVP）

- LV 电路 = 橡胶 + 板 + 电路板；缺任一项卡科技
- 群游戏：里程碑「组装机」解锁后，面板显示 **材料网**（GT 科技网味道）

## 四、融合玩法： enriched 流程

### 4.1 宏观三阶段（玩家体感）

```
阶段 I · 活下来（TFC 石器）
  探索 → 石工具 → 粘土/坑窑 → 发现铜锡粒

阶段 II · 铸起来（TFC 铸造）
  浇铸铜镐 → 青铜 → 勘矿 → 发现铁/煤脉
  并行：营养/季节/腐败开始有意义

阶段 III · 炼与机（TFC 铁器 × GT 蒸汽）
  【分叉】先高炉还是先蒸汽？（见 §4.3）
  高炉线 → 钢 + 砧锻
  蒸汽线 → 双倍矿 + 洗涤 + 硫副产
  合流 → 砖高炉 + 钢产量跃升
```

### 4.2 MVP 里程碑（v2，顺序可调）

| # | 里程碑 | 来源 | 解锁 |
|---|---|---|---|
| 1 | 石制工具 | TFC | 基础采集 |
| 2 | 坑窑 | TFC | 烧窑、熔化 |
| 3 | 模具+坩埚 | TFC | 浇铸 |
| 4 | 铜镐 | TFC | 采铜 |
| 5 | 青铜镐 | TFC | **铸造时代**、勘矿镐 |
| 6 | 铁矿脉（勘探） | TFC | 定向采铁 |
| 7 | 高炉（群项目） | TFC | 生铁 |
| 8 | 砧锻钢 | TFC | **铁器时代** |
| 9 | 蒸汽锅炉 | GT | steam 供给 |
| 10 | 蒸汽粉碎机 | GT | 粉碎链 |
| 11 | 洗矿台 | GT | 副产/硫 |
| 12 | 砖砌高炉 | GT | 高效钢 |
| 13+ | LV / 彩钢 | 扩展 | |

### 4.3 【新方向】铁器门口的分叉

调研认为：**这是避免「线性挂机」的关键设计**。

进入铁器时代前后，部落必须选 **主修方向**（可切换，有切换成本）：

| 方向 | 路径 | 优势 | 代价 |
|---|---|---|---|
| **🔥 冶炼派** | 先高炉+砧锻 | 快速钢工具；个人 tier ↑ | 矿石单次产量低 |
| **⚙️ 工业派** | 先蒸汽+粉碎机 | 矿石 ×2、副产硫 | 钢工具推迟；耗 bronze/steam |

- 选冶炼派：面板突出 `砧锻`、`木炭`
- 选工业派：面板突出 `添柴`、`粉碎`、`洗涤`
- **7 天后可免费改派一次**，之后改派耗 coin（经济联动，非 Pay2Win）

合流条件：同时拥有 **钢工具** + **洗矿台** → 解锁「砖砌高炉」

### 4.4 【新方向】部落工单（.async 协作）

大型节点（高炉、砖高炉）除材料捐献外，发 **工单** 到群面板：

```
📋 工单 #12 · 高炉地基
   还需：🪨×40  🧱×20  🔥木炭×30
   贡献：@小明 🪨×15  @小红 🔥×10
   [捐材料] ← 表情按钮
```

- 不强制在线同时点
- 贡献记入个人 ledger → coin/称号

### 4.5 【新方向】世界简报（持久沙盘心跳）

每 **现实日** 首次 `@求生` 时，若距上次 >12h，发 **世界简报**：

```
📰 世界简报
   季节：冬 → 补给消耗+1
   脉点「北坡铁矿」剩余 12%
   仓库：硫已达 40（洗矿线可用）
   工单 #12 进度 68%
```

让「沙盘在运转」可感知，不要求玩家每天打卡。

## 五、已定方向夯实清单

以下与此前讨论一致，本次调研**写入机制细节**，不再摇摆：

| 已定 | 夯实内容 |
|---|---|
| 持久化群存档 | `group_id` ↔ DB；每次行动写库；无退出保存 |
| 焦点 B | `@求生` 进 / `@退出` 出；只管路由 |
| 无回合/极高 TTL | 季节按现实周；矿脉随时间枯 |
| 表情面板 + Pillow | 主面板/科技树/副产看板/矿脉图 |
| 无背包 | → 改为 **6 槽工具栏** + 部落仓库 |
| 个人体力 | 高效次数/日；均衡度影响恢复 |
| 配方全透明 | 合金/加工/路线全显示 |
| 金币 | 里程碑奖、改派费、商人稀有副产；不买补给 |
| TFC+GT 分层 | 前期 TFC 铸造链；中期 GT 蒸汽处理；见 §4.1 |
| 不做 | 猜配比、物流、电气通关 |
