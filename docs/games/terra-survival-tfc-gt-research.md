# 群峦求生 · TFC / GT 调研笔记

- **Status**: Research v1
- **Last Updated**: 2026-05-21
- **用途**: 为 QQ 群持久化沙盘玩法提供**真实 mod 骨架**；内容数值后续在 MVP 阶段再定

---

## 一、调研结论（先看这个）

### 1.1 两个 mod 各自的核心是什么

| | 群峦传说 (TFC) | 格雷科技 (GTCEu) |
|---|---|---|
| **主轴** |  believable 生存 + 三大时代手工冶金 | 电压分层 + 矿物处理网 + 工业扩张 |
| **进度感** | 石器 → 铸造 → 铁器（再到彩钢） | 手搓 → 蒸汽 → LV → MV → HV … |
| **操作乐趣** | 塑形、坑窑、浇铸、砧上锤炼、勘矿 | 选处理路线、凑副产物、升电压、造多方块 |
| **资源压力** | 饥渴/温度/营养、工具分级、矿脉稀少 | 瓶颈材料（硫、硅、橡胶…）、蒸汽/EU 供给 |
| **不能搬到 QQ 的** | 3D 塑形图案、实时温度、重力、季节渲染 | 物流管道、电压网、产线布局、编程电路 |

### 1.2 群游戏应保留的「味道」

从两个 mod **各取一层**，不要混成一条假科技线：

```
TFC 层（前期～中期）：生存 + 勘矿 + 手工冶金时代跃迁
GT  层（中期～后期）：蒸汽动力 + 矿物多步处理 + 路线分叉
```

**趣味等价物**（不依赖空间/simulation）：

1. **时代门槛**：没青铜镐就不该高效采铁——工具是门票
2. **多步加工有仪式感**：坑窑烧 → 浇铸 → 锤炼，每步是面板选项，不是一键
3. **勘矿/开脉**：探索发现矿脉类型与储量，不是地上捡
4. **处理路线选择**：同种矿走「快而亏」还是「慢而副产多」
5. **部落瓶颈**：仓库里堆满 A 却缺 B，面板可见
6. **世界会变**：矿脉枯竭、季节修正（可选，真实时间）

### 1.3 明确不做

- 猜合金配比 / 隐藏合成表
- 合成格摆放、线路铺设、上下游物流设计
- 一局一 session、24h 超时、通关删档
- 个人背包格子（起步）；金币直接买食物

## 二、群峦传说 (TFC) 真实科技线

> 来源：[TFC 官方手册](https://terrafirmacraft.github.io/Field-Guide/18/zh_cn/)、TNG wiki

### 2.1 三大时代（官方定义）

| 时代 | 标志 | 关键能力 |
|---|---|---|
| **石器时代** | 开局 | 捡石子/木棍、石子塑形、石制工具；效率低 |
| **铸造时代** | 制陶 + 坑窑 + 浇铸 | 铜/青铜工具；**镐、勘矿镐、凿、锯**首次出现 |
| **铁器时代** | 锻铁 → 钢 → 彩钢 | 砧上锤炼、高炉、坩埚合金、黑/红/蓝钢 |

### 2.2 石器时代 → 铸造时代（真实步骤）

```
捡木棍/石子
  → 石子塑形（石刃/石镐头等，凿错会废料——群游戏可改为选模式，不玩图案）
  → 石制工具 + 木棍
  → 找地表矿粒（12 种，各对应金属）
  → 制陶：粘土塑形 → 坑窑烧制（~1400°C）
  → 模具（锭模/工具模）
  → 小缸/坩埚 + 坑窑熔化矿粒（≥100mB 才能浇铸）
  → 浇铸 → 第一把铜镐
  → 坩埚合金：青铜（铜+锡，比例在手册有但游戏内不直观——**群游戏应直接显示**）
  → 青铜工具 → 进入铸造时代实质玩法
```

**群峦特色机制**：

| 机制 | Mod 行为 | 群游戏可译 |
|---|---|---|
| 地表矿粒 | 10mB/粒，仅矿脉附近 15×35 格内 | 探索「发现矿脉」事件，解锁定向采集 |
| 坑窑 | 8干草+8原木，加热 6h 至黄白 | 建造「坑窑」里程碑，解锁烧陶/熔炼 |
| 浇铸 | 液态金属 → 模具 → 工具头 + 木棍 | 加工链：`熔化` → `浇铸` 两步 |
| 勘矿镐 | 25³ 区域，报告微量～巨量；有误报 | 「勘探」行动：耗体力，回报矿脉储量/类型 |
| 矿脉与岩层 | 不同矿石对应岩石类型与高度 | 简化：矿脉带「岩层标签」，错误区域效率低 |

### 2.3 铁器时代（真实步骤）

```
勘矿找到铁矿脉
  → 高炉（多方块：鼓风、木炭、助焊剂、铁矿石）
  → 产出生铁（Pig Iron）→ 坩埚浇铸生铁锭
  → 砧上锤炼：生铁 → 高碳钢 → 普通钢
  → 坩埚合金 → 脆钢/脆黑钢/脆蓝/脆红钢
  → 焊接 + 砧锤炼 → 黑钢 / 蓝钢 / 红钢
```

**彩钢配方（手册记载，群游戏仅作 flavor 参考，不做猜比例）**：

| 合金 | 坩埚成分（百分比） |
|---|---|
| 脆钢 | 钢 50–70%、镍 15–25%、黑铜 15–25% |
| 脆蓝钢 | 黑钢 50–55%、钢 20–25%、铋铜 10–15%、纹银 10–15% |
| 脆红钢 | 黑钢 50–55%、钢 20–25%、黄铜 10–15%、玫瑰金 10–15% |

**MVP 建议**：群游戏第一阶段**做到普通钢 + 高炉**即可；黑/红/蓝钢作为远期扩展。

### 2.4 TFC 生存维度（可选简化）

| 维度 | 作用 | 群游戏建议 |
|---|---|---|
| 饥饿/口渴 | 持续消耗，影响 max HP 与效率 | → **部落补给池** |
| 营养（6 种） | 影响生命上限 | → 简化为「均衡/偏科」buff，不做 six-track |
| 温度 | 衣物/火/季节 | → 季节事件修正探索/采集 |
| 技能 | 越用越快 | → **个人专长** + 工具 tier |

## 三、格雷科技 (GTCEu / GTNH) 真实科技线

> 来源：[GTCEu GitHub](https://github.com/GregTechCEu/GregTech)、[GTNH Steam Age](https://wiki.gtnewhorizons.com/wiki/Steam_Age)、[Ore Processing](https://wiki.gtnewhorizons.com/wiki/Ore_Processing)

### 3.1 宏观阶段

| 阶段 | 动力 | 标志 |
|---|---|---|
| 史前 | 手搓 / 原版炉 | 手打粉、熔炉烧矿 |
| **蒸汽时代** | 锅炉 → 蒸汽 (2 Steam : 1 EU) | 青铜基础设施、蒸汽机台 |
| **LV** | 32 EU/t 以下电力 | 电路、基础单方块机器 |
| **MV+** | 更高电压 | 洗矿、热力离心、电解、EBF… |

### 3.2 蒸汽时代（GTNH 真实目标链）

```
大量青铜（合金炉/蒸汽合金炉）
  → 小型燃煤锅炉 + 青铜管道（木管不能通蒸汽！）
  → 蒸汽机器：合金炉、压缩机、提取机、锻造锤、熔炉、粉碎机…
  → 锻造锤：高效产板/碎矿
  → 粉碎机：矿石产量 ×2（核心！）
  → 砖砌高炉 (Bricked Blast Furnace) → 钢
  → 高压蒸汽机台 → 仍比 LV 低效，但 several tiers 仍有用
```

**蒸汽特色**：不是「造个蒸汽机就完事」，而是 **锅炉供水 + 管道 + 机器排气口** 一整套——群游戏译为 **「蒸汽供给」资源 + 设施每回合耗蒸汽**。

### 3.3 矿物处理链（GT 灵魂）

典型矿石 **多路线、多副产物**（以铝土矿为例）：

| 路线 | 步骤 | 特点 |
|---|---|---|
| A | 粉碎 → 再粉碎 → 简单洗涤/离心 | 副产 Grossular |
| B | 粉碎 → 热力离心 → 粉碎 | 副产 Rutile + Gallium |
| C | 粉碎 → 洗涤 → 粉碎 → 洗涤/离心 | 更多 Rutile |

**自动流水线阶段**（Mod 里）：

```
原矿 → [锻造锤 1x / 粉碎机 2x] → 粉碎矿
     → [洗矿厂 / 炼药锅洗] → 纯净粉碎矿
     → [热力离心 / 离心机] → 不纯粉 / 纯粉
     → [粉碎机 / 离心机] → 最终粉 + 多种副产物
     → [电解/离心] → 元素级材料
```

**群游戏可译**（无物流）：

- 同一矿在加工面板选 **路线 A/B**（快而少副产 vs 慢而多副产）
- 副产物进部落仓库，后期科技需要「硫粉」「硅粉」等才会用到
- 机器有 **每时段处理次数上限**（/async 友好）

### 3.4 电压与材料网（远期）

- 机器按 **ULV / LV / MV / HV…** 分级；高 tier 电路需要 lower tier 产物交叉
- **EBF（电力高炉）**、**装配线** 等是里程碑建筑，非摆放
- 趣味在 **「缺一个 MV 电路就卡整条线」** → 面板显示瓶颈

**MVP 建议**：群游戏 GT 层 **做到蒸汽 + 粉碎机 + 洗矿 + 砖高炉** 即可；LV 电路作为扩展。

## 四、融合方案：群游戏 MVP 科技线（草案）

> 这是调研推导的 **第一版可玩骨架**，数值待 balance。

### 4.1 分层结构

```
┌─────────────────────────────────────────────────────────┐
│  TFC 层：生存 + 勘矿 + 坑窑/浇铸/砧锻 + 时代跃迁          │
├─────────────────────────────────────────────────────────┤
│  GT 层：蒸汽动力 + 粉碎/洗涤 + 处理路线 + 砖高炉          │
└─────────────────────────────────────────────────────────┘
```

### 4.2 里程碑列表（建议 MVP 范围）

| # | 里程碑 | 来源 | 解锁 |
|---|---|---|---|
| 1 | 石制工具 | TFC 石器 | 基础采集 |
| 2 | 坑窑 | TFC | 烧陶、熔化 |
| 3 | 铜工具（浇铸） | TFC 铸造 | 采铜效率 ↑ |
| 4 | 青铜工具 | TFC 铸造 | **铸造时代**；勘矿镐 |
| 5 | 勘探矿脉 | TFC | 定向采铁/煤 |
| 6 | 高炉 | TFC | 生铁 → 钢 |
| 7 | 砧锻钢 | TFC 铁器 | **铁器时代** |
| 8 | 蒸汽锅炉 | GT 蒸汽 | 蒸汽供给 |
| 9 | 蒸汽粉碎机 | GT | 矿石 ×2 路线 |
| 10 | 洗矿台 | GT | 副产物路线分叉 |
| 11 | 砖砌高炉 | GT | 高效产钢 |
| 12 | （扩展）LV 电力 | GT | 电路时代 |

**不做为 MVP**：彩钢、核聚变、装配线、信号塔式通关

### 4.3 群游戏专属机制（调研推导）

| 机制 | 说明 |
|---|---|
| 部落仓库 | 资源唯一池；加工/建造从仓库扣 |
| 个人工具 | 石/铜/青铜/铁/钢镐，决定采集 tier 与效率 |
| 个人体力 | 高效行动有限；超出低效但不封死 |
| 矿脉储量 | 脉点有 `remaining`；枯竭后需再勘探 |
| 处理路线 | 粉碎矿选 A/B，副产物不同 |
| 群项目 | 大型建筑（高炉/砖高炉）可设部落捐献进度条 |
| 焦点模式 | `@求生` 进入 / `@退出` 退出；存档始终保留 |
| Pillow 面板 | 科技树图 + 行动面板 + 仓库摘要 |
