## 零、定位

**群级持久化沙盘工具** — 有存档、无终局、无随意过期。

| 行为 | 含义 |
|---|---|
| `@机器人 求生` | 进入焦点 → 读档 → 发面板（可能带世界简报） |
| `@机器人 退出` | 退出焦点；**存档已在每次行动时写入** |
| 数据 | 挂在 `group_id`，与其他「一局游戏」解耦 |

融合：**TFC 生存与手工冶金** + **GT 蒸汽与矿物处理网**。  
内容细节以调研文档为准；本文定 **流程、架构、系统边界**。

---

## 一、设计原则（夯实）

1. **异步不限时** — 无回合计时；季节按现实周推进
2. **不强制多人** — 一人可推；工单/捐献/async 协作
3. **操作极简** — 主路径：Pillow 面板 + 表情回应
4. **乐趣在攀科技** — 配方全透明；瓶颈可见；**不猜配比**
5. **持久沙盘** — 矿脉枯、季节变、副产堆、工单进度持续
6. **NapCat 原生** — 表情当按钮；文字/数字兜底

## 二、核心体验

### 2.1 玩家在做什么

```
看面板（时代/里程碑/仓库/steam/补给）
  → 点表情选行动
  → 多步加工（烧窑/浇铸/粉碎/洗涤…）
  → 个人工具升级 + 部落设施升级
  → 时代跃迁 / 分叉选择 / 合流
  → 世界简报告诉你「不在的时候发生了什么」
```

### 2.2 宏观三阶段

| 阶段 | 来源 | 体感 |
|---|---|---|
| **I · 活下来** | TFC 石器 | 探索、石工具、粘土、坑窑 |
| **II · 铸起来** | TFC 铸造 | 浇铸铜/青铜、勘矿、营养/季节 |
| **III · 炼与机** | TFC 铁器 × GT 蒸汽 | 高炉/砧锻 **或** 蒸汽/粉碎；合流砖高炉 |

详见调研 §4.1、§4.3。

### 2.3 趣味从哪来（群聊可承载）

| 来源 | 群游戏等价 |
|---|---|
| TFC 制陶坑窑 | `烧窑` 行动链，非一键合成 |
| TFC 浇铸 | 熔化 → 浇铸 → 装柄 |
| TFC 勘矿 | 脉点、岩层、储量、误报 |
| TFC 砧锻 | 快锻/精锻抉择 |
| TFC 营养/季节 | 均衡度 + 现实周季节 |
| GT 蒸汽 | steam 池 + 添柴 + 机器耗汽 |
| GT 处理链 | 粉碎 → 洗涤 → 粉；路线 A/B |
| GT 副产物 | 硫/锡/银堆仓库；看板预警 |
| 群游戏层 | 工单、分叉、世界简报、个人工具栏 |

**不做**：合成格、拉线、猜配方、电气通关。

## 三、交互流程（已定）

### 3.1 焦点模式 B

| 指令 | 行为 |
|---|---|
| `@机器人 求生` | 注册路由；读档；发主面板；若 >12h 未登录发 **世界简报** |
| `@机器人 退出` | 注销路由 |
| `@机器人 面板` | 刷新（不耗行动） |

路由：焦点内处理面板表情 + 关键词；焦点外忽略旧面板表情。

### 3.2 面板层级

```
主面板 [🔍探索 ⛏️采集 🔥烧窑 🔨加工 🏗️建造 📋工单 📊科技 🔄刷新]
  └─ 子面板（加工/建造/采集/路线选择）→ 执行 → 刷新主面板
```

### 3.3 Pillow 页面

| 页面 | 内容 |
|---|---|
| 行动面板 | 时代、里程碑、补给、steam、仓库摘要 |
| 科技树 | TFC 时代 + GT 蒸汽枝；分叉状态 |
| 副产物看板 | 硫/锡/银… 堆叠与缺口（GT 味道） |
| 矿脉图 | 脉点位置/岩层/剩余% |
| 个人状态 | 工具栏 6 槽、体力、均衡度 |

## 四、架构与持久化（已定）

```
group_id ──→ TerraWorldRecord
              state_json
              updated_at
              schema_version
```

- 每次行动 `persist()`；不依赖 `GameSessionRecord.ended`
- `serialize_actions = True`（仓库互斥）
- 焦点状态在 `state.focus` 或独立内存表；**焦点丢失不丢档**
