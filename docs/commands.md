## 设计原则

1. **统一 @机器人**：避免机器人误响应普通群聊
2. **无斜杠**：所有命令不需要 / 前缀，直接 @机器人 + 关键词
3. **快捷开局优先**：每个游戏一个快捷词，一步直达
4. **命令无游戏前缀**：`结束` 而非 `汤 投降`，根据当前游戏状态自动判断
5. **提问无格式要求**：@机器人 说话就是提问（海龟汤）或抢答（趣味问答）
