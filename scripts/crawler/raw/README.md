## 质量初评（70 题主数据源）

**维度**：合理性 / 水平思考味 / 可改写性 / 清洗难度

- **优秀（可直接入库）**：约 25-30 题，如《1237》《外婆的苹果》《借书》《一巴掌》《五步》《不能回来》《天窗》《奶奶的橘子树》《一盘排骨》《1237》《贝尔公主》《灰姑娘》《人鱼公主》《午夜列车》等
- **中等（需要改写/补线索）**：约 30 题，主要是描述过于血腥或细节冗余
- **差（建议直接丢弃）**：约 10-15 题，逻辑漏洞大、反转牵强、或存在事实错误（如《洗头》机器人洗头不合理、《灵验》太简陋）

调性分布（主观）：
- **恐怖/血腥**：约 35 题（占 50%）——符合你放开禁令后的期待
- **悬疑/反转**：约 20 题（占 28%）
- **温情**：约 8 题（占 11%）——偏少，可能需要 LLM 补充
- **日常/奇幻**：约 7 题

⚠️ **预警**：这些题里有不少涉及**分尸/食人/精神异常**等重口内容（如《你妈真好》《奶奶》《饺子》《作家》《蝴蝶结》），符合你"内部群不设限"的定位，但如果未来要开放给更大群体需要过滤。
